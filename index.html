Getting Started

Welcome to the WeatherAPI documentation. To begin using our API, you'll need to sign up on our website to get your API key.

Follow these steps:

  1. Create an account on WeatherAPI.
  2. Go to your account settings and generate an API key.
  3. Once you have your API key, you're ready to start making requests to our API.

For detailed instructions, refer to our Authentication section.

Authentication

To make requests to the WeatherAPI, you need to include your API key in the query parameters of your HTTP request.

Example:


          GET /v1/current.json?key=YOUR_API_KEY&q=London HTTP/1.1
          Host: api.weatherapi.com

Replace YOUR_API_KEY with your actual API key obtained from your WeatherAPI account.

Endpoints

WeatherAPI provides various endpoints to access different types of weather data.

Examples:

  • Current Weather: /v1/current.json
  • Forecast Weather: /v1/forecast.json

Refer to our API Documentation for detailed information on each endpoint.

Examples

Here are some examples demonstrating how to use WeatherAPI's endpoints:

Example 1: Getting Current Weather

Replace YOUR_API_KEY with your actual API key and London with your desired location.
